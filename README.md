# Unemployment Analysis Using Python

##  Project Overview

This project analyzes unemployment trends using Python and data visualization techniques. The objective is to understand unemployment patterns across different states and time periods, identify key trends, and evaluate the impact of COVID-19 on unemployment rates. The project demonstrates the complete data analysis workflow, including data preprocessing, exploratory data analysis (EDA), visualization, and interpretation of results.

---

##  Objectives

- Analyze unemployment data using Python.
- Clean and preprocess the dataset.
- Perform Exploratory Data Analysis (EDA).
- Visualize unemployment trends across different states.
- Compare unemployment rates before and after COVID-19.
- Identify patterns and correlations within the dataset.
- Generate meaningful insights from real-world data.

---

##  Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn


3. Open the Jupyter Notebook:

```bash

```

4. Run **Unemployment_Analysis.ipynb**.
